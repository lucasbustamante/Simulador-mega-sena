# Mega‑Sena • API + Simulador 6×60

Projeto React (Vite) com Tailwind, pronto para rodar no VS Code.

## Rodar localmente

1) Entre na pasta do projeto
```bash
cd megasena-app
```

2) Instale as dependências
```bash
npm install
```

3) Inicie o server de dev
```bash
npm run dev
```

O app abrirá em http://localhost:5173

## Build
```bash
npm run build
npm run preview
```
